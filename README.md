# Florida Universities Social Media Analysis

## Overview
Tracked and analyzed Instagram and LinkedIn engagement metrics 
across 12 Florida universities over a 7-week period as part of 
a semester-long IT Leadership project (LIS 4480, FSU).

## Tools Used
- Microsoft Excel (pivot tables, SUMIF, conditional formatting)
- Data collection & trend analysis
- Data storytelling & presentation

## What I Did
- Collected engagement metrics (likes, shares, reach, posting 
  frequency) with Wednesday and Sunday checkpoints
- Built an Excel workbook with Raw Tracking, Summary, and 
  Dashboard tabs
- Identified top-performing content types and optimal posting 
  patterns per platform
- Delivered findings in a group presentation covering engagement 
  benchmarks, platform comparisons, and strategy recommendations

## Key Findings
- Instagram consistently outperformed LinkedIn in raw engagement
- Universities that posted collaboratively (tagging partners) 
  saw higher reach
- April Fools' campaigns (ex: University of Miami) drove 
  significant engagement spikes
